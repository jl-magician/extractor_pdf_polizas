**Create `policy_extractor/cli.py`** — a single-file Typer app. Structure:

```python
import typer
from rich.console import Console
from rich.progress import Progress, SpinnerColumn, BarColumn, TextColumn, MofNCompleteColumn, TimeElapsedColumn
from rich.table import Table
```

**App setup:**
```python
app = typer.Typer(name="poliza-extractor", help="Extract insurance policy data from PDFs.")
console = Console(stderr=True)  # Rich output to stderr
```

**`extract` command** (`@app.command()`):
- Parameters per CONTEXT.md locked decisions:
  - `file: Path = typer.Argument(..., help="Path to PDF file", exists=True)`
  - `model: str = typer.Option(None, "--model", help="Override extraction model")`
  - `force: bool = typer.Option(False, "--force", help="Reprocess even if already extracted")`
  - `output_dir: Optional[Path] = typer.Option(None, "--output-dir", help="Write JSON to directory")`
  - `verbose: bool = typer.Option(False, "--verbose", help="Show detailed output")`
  - `quiet: bool = typer.Option(False, "--quiet", help="Suppress all output except JSON")`
- Flow:
  1. Call `init_db(settings.DB_PATH)` and `SessionLocal.configure(bind=engine)`
  2. Open session with `SessionLocal()`
  3. Compute file hash with `compute_file_hash(file)`
  4. If not `force`: check `is_already_extracted(session, file_hash)` — if True, print `[SKIP] {file.name} -- already extracted` to stderr and `raise typer.Exit(0)`
  5. Call `ingest_pdf(file, session=session, force_reprocess=force)`
  6. Call `extract_policy(ingestion_result, model=model)` — destructure as `(policy, usage)`
  7. If `policy` is None: print error to stderr, `raise typer.Exit(1)`
  8. Print `policy.model_dump_json(indent=2)` to stdout via `print()`
  9. If `output_dir`: create dir, write JSON to `output_dir / f"{file.stem}.json"`
  10. If `usage` and not `quiet`: print cost info to stderr using `estimate_cost`
  11. Format: `"Tokens: {input_tokens:,} input, {output_tokens:,} output | Est. cost: ${cost:.4f} USD"`

**`batch` command** (`@app.command()`):
- Same parameters as `extract` except `folder: Path = typer.Argument(...)` instead of `file`
- Validate folder exists and is directory; if not, print error to stderr and exit 1
- Enumerate PDFs: `sorted(folder.glob("*.pdf"))` — if empty, print warning and exit 0
- Initialize counters: `succeeded = 0, failed = 0, skipped = 0, total_input = 0, total_output = 0`
- Initialize `failures: list[tuple[str, str]]` for (filename, error_msg)
- DB setup same as extract command
- Open session, then wrap file loop in Rich Progress context manager:
  ```python
  with Progress(
      SpinnerColumn(),
      TextColumn("[progress.description]{task.description}"),
      BarColumn(),
      MofNCompleteColumn(),
      TextColumn("{task.percentage:>3.0f}%"),
      TimeElapsedColumn(),
      console=console,  # stderr
      disable=quiet,
  ) as progress:
  ```
- For each PDF in the progress loop:
  - Update progress description to `f"[cyan]{pdf.name}[/cyan]"`
  - Compute hash, check idempotency (skip if already extracted and not force)
  - On skip: increment `skipped`, advance progress, continue
  - Try: ingest + extract. On success: increment `succeeded`, accumulate tokens from usage
  - On exception: increment `failed`, append to `failures`, log warning, continue
  - If `output_dir` and policy: write JSON file
  - Advance progress
- After loop, use `time.time()` for elapsed time calculation (capture start before loop)
- Print Rich summary table to stderr with: Processed (total PDFs), Succeeded, Failed, Skipped, Total Time, Input Tokens, Output Tokens, Est. Cost (USD)
  - Use `estimate_cost` with the effective model (model param or settings.EXTRACTION_MODEL)
- If failures: print a second table listing each failed file and its error reason
- Exit code: 0 if no failures, 1 if any failures
