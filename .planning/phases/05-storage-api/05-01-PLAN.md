       - Write all tests from the behavior list above

    3. Create `policy_extractor/storage/writer.py`:
       - `upsert_policy(session: Session, extraction: PolicyExtraction) -> Poliza`:
         - Query existing by `filter_by(numero_poliza=extraction.numero_poliza, aseguradora=extraction.aseguradora).first()`
         - If found: clear `poliza.asegurados` and `poliza.coberturas`, then `session.flush()`
         - If not found: create new `Poliza(numero_poliza=..., aseguradora=...)`  and `session.add(poliza)`
         - Set scalar fields via setattr loop: tipo_seguro, fecha_emision, inicio_vigencia, fin_vigencia, nombre_contratante, nombre_agente, prima_total, moneda, forma_pago, frecuencia_pago, source_file_hash, model_id, prompt_version, extracted_at
         - Set campos_adicionales: merge extraction.campos_adicionales with `{"confianza": extraction.confianza}` (store confianza inside campos_adicionales)
         - Append Asegurado rows from extraction.asegurados (map all fields)
         - Append Cobertura rows from extraction.coberturas (map all fields)
         - `session.commit()` and return poliza

       - `orm_to_schema(poliza: Poliza) -> PolicyExtraction`:
         - Reconstruct PolicyExtraction from ORM columns
         - Extract confianza from `poliza.campos_adicionales.get("confianza", {})` back into the top-level `confianza` field
         - Build campos_adicionales WITHOUT the "confianza" key (it goes to the dedicated field)
         - Map each poliza.asegurados to AseguradoExtraction
         - Map each poliza.coberturas to CoberturaExtraction

    4. Update `policy_extractor/storage/__init__.py` to export `upsert_policy` and `orm_to_schema`.

    5. Run `pytest tests/test_storage_writer.py -x -q` — all tests must pass.
