       ```python
       @dataclass
       class EvaluationResult:
           score: float           # (completeness + accuracy + (1 - hallucination_risk)) / 3

           evaluated_at: datetime
           model_id: str
           usage: anthropic.types.Usage
       ```

    3. Create `EVAL_SYSTEM_PROMPT` — instruct Sonnet to:
       - Compare extracted field values against PDF text character by character for key fields (numero_poliza, prima_total, fechas)
       - Check for fields visible in the PDF that Haiku left as null
       - Flag any value that does not appear verbatim or calculably in the source text
       - Score completeness 0.0-1.0 (fraction of visible fields captured)
       - Score accuracy 0.0-1.0 (plausibility/correctness of values)
       - Score hallucination_risk 0.0-1.0 (fraction of fields with invented data, 0=none)
       - Write in Spanish for field-level issues since domain terms are Spanish

    4. Create `build_evaluation_tool() -> dict` mirroring `build_extraction_tool()`:
       - name: EVAL_TOOL_NAME
       - input_schema with properties: completeness (number), accuracy (number), hallucination_risk (number), flags (array of {field: string, issue: string}), summary (string)
       - All 5 properties in required list

    5. Create `call_evaluation_api(client, assembled_text, extraction_json, model=EVAL_MODEL_ID, max_tokens=1024) -> anthropic.types.Message`:
       - Mirror `call_extraction_api` exactly
       - system=EVAL_SYSTEM_PROMPT
       - user message: "PDF TEXT:\n{assembled_text}\n\nEXTRACTED DATA:\n{extraction_json}"
       - tools=[build_evaluation_tool()]
       - tool_choice={"type": "tool", "name": EVAL_TOOL_NAME}

    6. Create `_parse_evaluation(message) -> EvaluationResult`:
       - Parse tool_use response: `dict(message.content[0].input)`
       - Clamp each score to [0.0, 1.0] range
       - Calculate overall score: `(completeness + accuracy + (1.0 - hallucination_risk)) / 3.0`
       - Build evaluation dict: `{"completeness": ..., "accuracy": ..., "hallucination_risk": ..., "flags": [...], "summary": "..."}`
       - Serialize to JSON string via `json.dumps(eval_dict, ensure_ascii=False)`
       - Return EvaluationResult with evaluated_at=datetime.now(timezone.utc), model_id=message.model

    7. Create `evaluate_policy(ingestion_result: IngestionResult, policy: PolicyExtraction, model: str = EVAL_MODEL_ID) -> EvaluationResult | None`:
       - Import assemble_text from extraction.prompt (lazy or top-level, since this module is itself lazily imported)
       - `assembled_text = assemble_text(ingestion_result)`
       - `extraction_json = policy.model_dump_json(indent=2)`
       - Create Anthropic client: `anthropic.Anthropic(api_key=settings.ANTHROPIC_API_KEY)`
       - Call `call_evaluation_api(client, assembled_text, extraction_json, model)`
       - Parse with `_parse_evaluation(message)`
       - Wrap in try/except Exception, log error and return None on failure
