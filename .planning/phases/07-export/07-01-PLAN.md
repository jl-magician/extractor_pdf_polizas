    Functions:
    - `_cell_value(val)`: if `isinstance(val, Decimal)` return `float(val)`, else return val as-is. Never convert date to string.
    - `_collect_extra_keys(items, attr="campos_adicionales")`: Two-pass: collect all unique keys from `getattr(item, attr) or {}` across all items. Strip "confianza" key. Return `list[str]` of unique keys in insertion order.
    - `_write_sheet(ws, items, base_columns, get_row_fn, extra_keys)`: Generic sheet writer. Writes header row (base_columns + extra_keys). Appends data rows using get_row_fn(item) for base columns and extras.get(key, "") for extra keys. Calls `_apply_formats(ws, header)` and `_finalize_sheet(ws)`.
    - `_apply_formats(ws, header_row: list[str])`: Iterate columns; for DATE_COLS apply DD/MM/YYYY number_format to all data cells; for MONEY_COLS apply #,##0.00. Use `get_column_letter` to iterate cells per column.
    - `_finalize_sheet(ws)`: If `ws.max_row > 1`, set `ws.auto_filter.ref = ws.dimensions`. Always set `ws.freeze_panes = "A2"`.
    - `export_xlsx(polizas: Sequence[Poliza], output_path: Path) -> int`: Creates Workbook, writes 3 sheets, saves. Returns count of polizas exported. Lazy-imports openpyxl inside the function body.
    - `export_csv(polizas: Sequence[Poliza], output_path: Path) -> int`: Opens file with `encoding='utf-8-sig'` and `newline=''`. Uses csv.DictWriter. Writes polizas only (flat). Returns count.

    For the polizas sheet get_row_fn: read base columns via `getattr(poliza, col)` through `_cell_value()`. For extras, read `poliza.campos_adicionales or {}`, strip confianza, return extras dict.

    For asegurados sheet: iterate `poliza.asegurados` for each poliza, inject `numero_poliza` from parent poliza.

    For coberturas sheet: iterate `poliza.coberturas` for each poliza, inject `numero_poliza` from parent poliza.

    Handle empty polizas list: write header-only sheets (no error), return 0.

    Handle PermissionError on wb.save() / file open: catch `PermissionError` and `OSError`, raise a new `ExportError(f"Cannot write to {output_path} — close the file if it is open in another application")`. Define `class ExportError(Exception): pass` at module level.
