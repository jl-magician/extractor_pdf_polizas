    The subcommand signature:
    ```python
    @app.command(name="batch-fixtures")
    def batch_fixtures(
        pdf_dir: Path = typer.Argument(..., help="Directory containing real PDFs", exists=True),
        output: Path = typer.Option(
            Path("tests/fixtures/golden"), "--output", "-o",
            help="Output directory for fixture JSON files"
        ),
        insurer_map: Optional[Path] = typer.Option(
            None, "--insurer-map",
            help="JSON file mapping PDF filename patterns to {insurer, type} slugs"
        ),
        model: Optional[str] = typer.Option(None, "--model", help="Override extraction model"),
        run_tests: bool = typer.Option(False, "--run-tests", help="Run pytest -m regression after creating fixtures"),
    ) -> None:
    ```

    Implementation logic:
    1. Discover all `*.pdf` files in `pdf_dir` (non-recursive, sorted)
    2. If `--insurer-map` is provided, load the JSON file. Format: `{"filename_pattern": {"insurer": "zurich", "type": "auto"}}`. Patterns are substring matches against the PDF filename (case-insensitive).
    3. If no insurer-map, attempt to infer insurer from PDF filename using `_infer_insurer(filename)` helper that checks if any of the 10 known insurer slugs (`zurich`, `qualitas`, `mapfre`, `axa`, `gnp`, `chubb`, `ana`, `hdi`, `planseguro`, `prudential`) appears as a substring in the lowercase filename. Type defaults to `"general"`.
    4. For each PDF:
       a. Call `ingest_pdf(pdf_path, session=session)`
       b. Call `extract_policy(ingestion_result, model=model)` — returns `(policy, usage, retries)`
       c. If policy is None: print warning via `console.print(f"[yellow]SKIP:[/yellow] {pdf.name} — extraction failed")`, increment skip counter, continue
       d. Call `policy.model_dump(mode="json")`
       e. Add metadata: `raw["_source_pdf"] = pdf.name`, `raw["_insurer"] = insurer_slug`, `raw["_tipo_seguro"] = type_slug`, `raw["_created_at"] = datetime.now(timezone.utc).isoformat()`
       f. Call `PiiRedactor().redact(raw)` on the result
       g. Determine sequence number: count existing files matching `{insurer}_{type}_*.json` in output dir, next seq = count + 1, zero-padded to 3 digits
       h. Write to `output / f"{insurer}_{type}_{seq:03d}.json"` with `json.dumps(redacted, indent=2, ensure_ascii=False, default=str)`
       i. Print `console.print(f"[green]OK:[/green] {pdf.name} -> {out_file.name}")`
    5. After all PDFs: print summary table via Rich showing total, succeeded, skipped counts
    6. If `--run-tests` flag: run `subprocess.run(["pytest", "-m", "regression", "-v"], check=False)` and print result

    The `_infer_insurer` helper function (module-level, above the batch-fixtures command):
    ```python
    _KNOWN_INSURERS = [
        "zurich", "qualitas", "mapfre", "axa", "gnp",
        "chubb", "ana", "hdi", "planseguro", "prudential",
    ]

    def _infer_insurer(filename: str) -> str:
        """Infer insurer slug from PDF filename. Returns 'unknown' if no match."""
        lower = filename.lower()
        for slug in _KNOWN_INSURERS:
            if slug in lower:
                return slug
        return "unknown"
    ```
