- `_NUMERIC_COLS`: frozenset containing `{"prima_total", "suma_asegurada", "deducible"}`
- `_DATE_COLS`: frozenset containing `{"fecha_emision", "inicio_vigencia", "fin_vigencia", "fecha_nacimiento"}`
- `_NON_NULLABLE_COLS`: frozenset containing `{"numero_poliza", "aseguradora", "tipo", "nombre_descripcion", "nombre_cobertura"}`

- `_coerce_value(col: str, raw_value: str)` function:
  - If `raw_value` is empty string and col not in `_NON_NULLABLE_COLS`: return `None`
  - If `raw_value` is empty string and col in `_NON_NULLABLE_COLS`: raise `HTTPException(422, detail="Campo obligatorio")`
  - If col in `_NUMERIC_COLS`: try `Decimal(raw_value)`, except `InvalidOperation` raise `HTTPException(422, detail="Valor numerico invalido")`
  - If col in `_DATE_COLS`: try `date.fromisoformat(raw_value)`, except `ValueError` raise `HTTPException(422, detail="Fecha invalida (YYYY-MM-DD)")`
  - Otherwise: return `raw_value` as-is (string)
  - Import `date` from `datetime` module at top

- `_apply_field_update(db, poliza, field_path: str, raw_value: str)` function per RESEARCH Pattern 5:
  - Split field_path on "."
  - len==1: top-level poliza column. Get old_value via `getattr(poliza, col, None)`, coerce via `_coerce_value`, setattr on poliza. Convert old_value to str for correction log (or None).
  - parts[0]=="campos_adicionales" and len==2: JSON key update. `current = dict(poliza.campos_adicionales or {})`, get old, set new, reassign full dict to poliza.campos_adicionales (MUST reassign for dirty-tracking, per Pitfall 1).
  - parts[0] in ("asegurados","coberturas") and len==3: nested row. Parse int(parts[1]) as row_id, db.get(Asegurado or Cobertura), verify obj.poliza_id == poliza.id, getattr old, _coerce_value, setattr.
  - Otherwise: raise HTTPException(400, "Campo desconocido: {field_path}")
  - Return (str(old_value) if old_value is not None else None, raw_value)

- `_field_label(field_path: str) -> str`: returns the last segment of the dot-path, replacing underscores with spaces, title-cased. E.g. "prima_total" -> "Prima Total", "asegurados.42.nombre_descripcion" -> "Nombre Descripcion".

- `_input_type(field_path: str) -> str`: extract the column name (last dot segment). If in `_NUMERIC_COLS` return "number", if in `_DATE_COLS` return "date", else "text".
